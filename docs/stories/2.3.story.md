# Story 2.3 — Resultado Financeiro em Tempo Real

**Epic:** EPIC-02 — Base Financeira  
**Status:** Done  
**Agente:** @dev  
**Esforço estimado:** M  
**Dependências:** Stories 2.1 e 2.2 concluídas

---

## User Story

Como vendedor,  
quero ver imediatamente o custo total, margem, lucro e ROI do produto,  
para que eu entenda a viabilidade financeira antes de definir o preço.

---

## Acceptance Criteria

- [x] **AC1:** Painel de resultados exibe: Custo Total (R$), Margem (%), Lucro (R$), ROI (%)
- [x] **AC2:** Exibe Break-even (quantidade de unidades para cobrir custo fixo mensal)
- [x] **AC3:** Exibe Preço Mínimo Viável (preço onde margem = 0%)
- [x] **AC4:** Todos os valores atualizam em tempo real (sem botão "calcular")
- [x] **AC5:** Indicador visual de saúde: verde (≥20%), amarelo (10-19%), vermelho (<10%)
- [x] **AC6:** Fórmulas de cálculo exibíveis em tooltip para transparência
