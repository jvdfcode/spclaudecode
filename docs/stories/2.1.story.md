# Story 2.1 — Formulário de Entrada de Custos + Resultado em Tempo Real

**Epic:** EPIC-02 — Base Financeira  
**Status:** Done  
**Agente:** @dev  
**Esforço estimado:** M  
**Dependências:** Stories 1.4 e 2.0 concluídas

---

## User Story

Como vendedor,  
quero preencher os custos de um produto e ver imediatamente se é viável vender no ML,  
para que eu tome decisões de precificação com dados reais sem usar planilha.

---

## Acceptance Criteria

- [ ] **AC1:** Formulário com todos os campos de `ViabilityInput`: CMV, embalagem, imposto, overhead, margem-alvo, preço de venda, tipo de anúncio, parcelas
- [ ] **AC2:** Cálculo acontece em tempo real (sem botão submit) a cada mudança de campo
- [ ] **AC3:** Painel de resultados exibe: Custo Total, Lucro, Margem%, ROI%, Preço Mínimo Viável, Preço Recomendado
- [ ] **AC4:** Badge de classificação colorido: verde (viável), amarelo (atenção), vermelho (não viável)
- [ ] **AC5:** Detalhamento de custos expansível mostrando cada componente
- [ ] **AC6:** Valores monetários formatados em BRL (R$ 1.234,56)
- [ ] **AC7:** `pnpm build` e `pnpm typecheck` limpos
