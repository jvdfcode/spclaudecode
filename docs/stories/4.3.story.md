**Dependências:** Story 4.2 concluída

## User Story
Como vendedor, quero abrir um SKU e ver o histórico de todos os cálculos feitos para ele, para que eu entenda como a margem evoluiu ao longo do tempo.

## Acceptance Criteria
- [x] AC1: Página de detalhe acessível via card na Central de SKUs
- [x] AC2: Exibe nome, status, notas e data de criação do SKU
- [x] AC3: Histórico de cálculos em ordem cronológica decrescente
- [x] AC4: Cada cálculo mostra: data, preço de venda, margem, ROI e tipo de anúncio
- [x] AC5: Botão "Recalcular" que abre a calculadora com os dados desse SKU pré-preenchidos
- [x] AC6: Botão voltar para a Central de SKUs
