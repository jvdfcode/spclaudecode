## User Story
Como vendedor, quero ver todos os meus SKUs salvos em uma grade com nome, margem, preço e badge de status colorido, para que eu tenha visão consolidada do meu portfólio.

## Acceptance Criteria
- [x] AC1: Grid de cards com nome, margem (%), preço de venda, badge de status colorido
- [x] AC2: Status: Viável (verde), Atenção (amarelo), Não viável (vermelho), Rascunho (cinza)
- [x] AC3: Filtro por status (tabs ou botões)
- [x] AC4: Busca por nome
- [x] AC5: Estado vazio amigável quando não há SKUs
- [x] AC6: Dados carregados do Supabase via Server Component
