## User Story
Como vendedor, quero buscar produtos similares ao meu no Mercado Livre, para que eu veja os preços reais praticados no mercado.

## Acceptance Criteria
- [x] AC1: Campo de busca por palavra-chave
- [x] AC2: Resultados buscados server-side (API Route Next.js → ML API)
- [x] AC3: Exibe até 50 anúncios por busca
- [x] AC4: Cada resultado mostra: título, preço, frete grátis, Full, reputação, vendidos
- [x] AC5: Loading state durante a busca
- [x] AC6: Mensagem clara se API ML retornar erro ou nenhum resultado
- [x] AC7: Cache no Supabase por 1 hora para mesma query
