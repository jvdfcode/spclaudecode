## User Story
Como vendedor, quero salvar um produto calculado como SKU com um nome, para que eu possa consultá-lo depois sem precisar redigitar tudo.

## Acceptance Criteria
- [x] AC1: Botão "Salvar como SKU" aparece na calculadora após preencher custo + preço de venda
- [x] AC2: Modal/inline com campo de nome obrigatório (notas opcionais)
- [x] AC3: SKU persistido no Supabase (tabelas `skus` + `sku_calculations`)
- [x] AC4: RLS ativo — usuário só acessa seus próprios SKUs
- [x] AC5: Após salvar, feedback visual de sucesso
- [x] AC6: Dados de cálculo completos salvos em `sku_calculations.cost_data` e `result_data`
