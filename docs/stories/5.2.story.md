## User Story
Como vendedor, quero que kits, combos e anúncios irrelevantes sejam removidos automaticamente, para que minha base de comparação seja confiável.

- [x] AC2: Filtro de condição: novo / usado / todos
- [x] AC3: Filtro manual de frete grátis
- [x] AC4: Deduplicação por preço + título normalizado
- [x] AC5: Índice de confiança: "Base limpa: X de Y anúncios (Z%)"
- [x] AC6: Usuário pode incluir/excluir anúncios manualmente
