## User Story

Como vendedor,  
quero criar conta e fazer login com email e senha,  
para que meus dados de SKUs e cálculos sejam salvos de forma segura e privada.

---

## Contexto Técnico

Implementa o sistema completo de autenticação usando Supabase Auth. O fluxo usa cookies httpOnly gerenciados pelo pacote `@supabase/ssr` — o middleware Next.js verifica a sessão em cada request e redireciona usuários não autenticados.

**Fluxo completo:**
1. Usuário acessa rota protegida → middleware verifica cookie de sessão
2. Sem sessão → redirect para `/login`
3. Login/cadastro → Supabase retorna JWT → cookie setado automaticamente
4. Rota protegida liberada

**Route groups (App Router):**
- `(auth)` — páginas de login/cadastro (sem navbar)
- `(app)` — páginas protegidas (com navbar)
