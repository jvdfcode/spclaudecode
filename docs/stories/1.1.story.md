# Story 1.1 — Setup do Projeto Next.js + Supabase

**Epic:** EPIC-01 — Fundação & Infraestrutura  
**Status:** Done  
**Agente:** @dev  
**Esforço estimado:** M (meio dia)  
**Dependências:** nenhuma — primeira story do projeto

---

## User Story

Como desenvolvedor,  
quero um projeto Next.js 14 configurado com TypeScript, Tailwind, shadcn/ui e Supabase conectado,  
para que o squad tenha uma base de desenvolvimento padronizada e funcional a partir da qual todas as outras stories possam ser construídas.

---

## Contexto Técnico

Esta é a story fundacional. Ela cria o esqueleto do projeto dentro da pasta `SmartPreço/` existente, que atualmente contém apenas documentação. O projeto Next.js será inicializado com `create-next-app` e configurado conforme a arquitetura aprovada em `docs/architecture/architecture.md`.

**Stack a instalar:**
- Next.js 14+ com App Router
- TypeScript 5+
- Tailwind CSS 3+
- shadcn/ui (tema base)
- Supabase JS client (`@supabase/ssr` + `@supabase/supabase-js`)
- Vitest (testes unitários)
- ESLint + Prettier

**Package manager:** pnpm (conforme ADR-04 da arquitetura)

---
