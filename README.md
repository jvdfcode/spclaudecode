# SmartPreço

> Sistema de apoio à decisão de preço para marketplace (Mercado Livre)

Baseado em custo total real, simulação de cenários e comparação com mercado real.

## Módulos

- **Base Financeira** — custo, margem, lucro, ROI, break-even, preço mínimo
- **Simulador de Cenários** — faixas de preço, posicionamento equilibrado
- **Central de SKUs** — portfólio, histórico, status por produto
- **Bloco Mercado** — anúncios reais ML, filtros, indicadores de confiança
- **Decisão de Preço** — integração dos módulos → preço vendável e sustentável

## Stack

- **Frontend:** Next.js + Tailwind CSS
- **Backend:** Node.js + TypeScript
- **Banco:** Supabase
- **Deploy:** Vercel (frontend) + Railway (backend)
